Preamble

Context('Matrix');

$A = Matrix([
    [ non_zero_random(-5, 5), non_zero_random(-5, 5) ],
    [ non_zero_random(-5, 5), non_zero_random(-5, 5) ]
]);

$B = Matrix([
    [
        non_zero_random(-5, 5),
        non_zero_random(-5, 5),
        non_zero_random(-5, 5)
    ],
    [
        non_zero_random(-5, 5),
        non_zero_random(-5, 5),
        non_zero_random(-5, 5)
    ],
]);

$rma1 = RadioMultiAnswer(
    [
        ['The product \(AB\) does not exist.'],
        [ 'The product \(AB\) is %s*', $A * $B ]
    ],
    1
);

$rma2 = RadioMultiAnswer(
    [
        ['The product \(BA\) does not exist.'],
        [ 'The product \(BA\) is %s*', $A * $B ]
    ],
    0
);

Setup

First, the two matrices are defined.

The RadioMultiAnswer method produces a set of radio buttons for each of the given statements. The format is

RadioMultiAnswer([
    [statement1],
    [statement2],
    ...
    [last statement]
], correct index, options)

Answer blanks can be added with the %s in the string and if a matrix is desired, %s* should be used. See the parserRadioMultiAnswer.pl for more details.
